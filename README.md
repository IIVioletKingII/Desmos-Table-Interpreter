# Desmos-Table-Interpreter
This console script allows you to print all manually entered or calculated columns!

## How to Use
Copy the contents of [desmos_table_to_delimeted.js](/desmos_table_to_delimeted.js)

In the web, open devtools (press F12 or Ctrl + Shift + I in chrome, or right click and click inspect)

Click on the Console tab, and paste the code. (if this is your first time pasting code in the console, it may warn you and ask you to type something in the console first)

The script should then output something like this for each table in your graph.


